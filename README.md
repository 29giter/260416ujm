中华人民共和国政府，特别是中国共产党政法委系统（包括武警）、中共军方系统[166]，被指控大规模系统性活体摘取良心犯的器官，供商业性移植给中国人或外国人谋利[167][168][169]，被害者因此死亡，主要对象是遭关押的法轮功修炼者[170][171][172]，还包括后来增加的 家庭教会基督徒、西藏人、维吾尔族人、持不同政见者，这些良心犯被关押在监狱、“劳动教养所”（亦即“劳动改造”、劳改营）等地。但中华人民共和国政府否认这项指控，也同时拒绝了外国组织独立调查的请求、拒绝公布相关数据。[173]

2019年在英国伦敦的独立人民法庭，由前国际刑事法庭检察官尼斯爵士主持，宣判认定中国共产党当局多年来在全国范围大量强摘活体器官，成立反人类罪，受害者主要是法轮功学员[174]。

联合国人权高级专员办公室2021年6月发布，12名联合国独立人权专家包括至少8位特别报告员联合声明，对于在中国被拘留的法轮功修炼者等少数群体遭到活摘器官的许多指控报告，感到极震惊；专家团指出，早在2006、2007年，联合国人权专家就曾向中国共产党当局提出强行活摘囚犯器官的担忧，但遗憾北京当时的回应缺乏相关数据，例如器官分配等待时间、或器官来源资讯等数据；专家团呼吁中方迅速回应活摘器官的指控，并允许国际人权机制进行独立监督。专家团指出，他们收到了可信的信息，受害群体遭到任意拘押，并被强制性验血与器官检查，但这些检查并非是对所有群体的例行公事，这些检查结果被登记在一个提供器官移植的数据库
